# <p align="center"> Story </p>

## Description
"Story" is a Kotlin-based plugin for Minecraft servers, designed to enhance player interactions through dynamic conversations, quests, and NPC management. It utilizes integrations with popular Minecraft plugins and APIs to provide an immersive and interactive game experience.

## Features
- **Dynamic NPC Conversations**: Manage and interact with NPCs seamlessly.
- **Quest Management**: Define and track main, side, daily, and event quests.
- **Lore and Context**: Generate and manage lorebooks for in-game storytelling.
- **Relationship Management**: Track and handle relationships between players and NPCs.
- **Location Services**: Manage and interact with in-game locations.
- **Plugin Integrations**:
  - PaperMC
  - Citizens NPC API
  - MythicMobs
  - PlaceholderAPI
  - DecentHolograms
  - ItemsAdder
  - BetterHealthBar
- **Command Utilities for Players and Admins**: Includes commands for toggling NPC behaviors, feeding messages, and managing conversations.
